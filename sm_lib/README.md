SM Lib - Proclame la bonne parole de Sam & Max
==========================================================================

Ce module sm_lib sert a proclamer la parole de Sam & Max, puissent-t-ils 
retouver une totalité liberté de pensé consmique

pip install sm_lib

    >>> proclamer()
